## Setup

Clone the repo
```js
git clone https://github.com/ariefyusron/urbanhire-backend-test-with-express.git <project_name>
```

Open directory
```js
cd <project_name>
```

and then run
```js
npm install
```


## Start Server

```js
npm start
```
